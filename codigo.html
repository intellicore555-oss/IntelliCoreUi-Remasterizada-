Tecnologias utilizadas no app

Smali
Linguagem usada no nível baixo de apps Android. É o código que aparece após descompilar APKs e permite manipulação direta da lógica interna do aplicativo.
Kotlin
Linguagem oficial do Android usada para criar o app. Moderna, segura e totalmente integrada com o sistema Android.
HTML, CSS e JS
Usados para criar páginas internas, interfaces web e possíveis telas híbridas dentro do aplicativo.
XML
Responsável pelos layouts do Android, como telas, botões, listas e widgets estruturais do app.
SVG
Formato de imagens vetoriais usado para ícones. Ele mantém qualidade perfeita em qualquer tamanho sem perder nitidez.
